AMAZON WEB SCRAPER

This project is a Python-based web scraper that extracts product details from Amazon pages.
It demonstrates practical use of web scraping techniques for data collection and analysis.

FEATURES

Scrapes product title, price, rating, and availability

Uses Python libraries for HTTP requests and HTML parsing

Clean and simple notebook-based implementation

TECH STACK

Python

BeautifulSoup

Requests

Jupyter Notebook

USE CASE

Useful for learning web scraping, price tracking, and basic data analysis workflows.
